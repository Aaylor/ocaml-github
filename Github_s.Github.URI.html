Module Github_s.Github.URI

module URI: sig .. end
Useful URI generation functions, normally for displaying on a web-page. The Github_s.Github.URI.authorize function is the entry URL for your users, and the Github_s.Github.URI.token is the URI used to convert the result into a concrete access token.
